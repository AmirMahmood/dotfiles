## Installation
### Git
* Install [git-credential-manager][4] and choose your [credential stores][5]
* set `user.name` nad `user.email` configs

### NeoVim
* Follow [this page][6] for Neovim installation.
* Then install [packer.nvim][7] plugin manager.
